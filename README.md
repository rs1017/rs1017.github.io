# Printable Playroom

유아/초등 저학년을 위한 **이미지 중심 프린터블 활동지 갤러리** 저장소입니다.

이 프로젝트는 긴 글보다 **바로 보고 출력할 수 있는 이미지 세트**를 우선합니다.

## 사이트 방향

- 홈: 블로그 피드가 아닌 갤러리형 화면
- 포스트: 한 주제당 최소 10장 이미지
- 이미지 타일: 이미지 1장 + 짧은 제목 1개
- 설명문: 갤러리/아트 페이지에는 장문 설명을 넣지 않음

## 현재 컬렉션

- 봄 색칠 세트 (`spring-coloring-pack.html`)
- 바다 오리기/붙이기 세트 (`ocean-cut-paste.html`)
- 정원 숨은그림 세트 (`garden-hidden-picture.html`)

## 콘텐츠 제작 규칙

1. 한 포스트 = 한 테마
2. 제작 전 10컷 샷리스트 먼저 작성
3. 최소 10개 이미지 확정 후 페이지 구성
4. 이미지마다 짧은 제목 부여
5. 페이지는 이미지 컬렉션처럼 보이도록 유지

상세 워크플로우는 `IMAGE_WORKFLOW.md`를 참고하세요.

## 로컬 실행

```bash
bundle install
bundle exec jekyll serve
```
