# Printable Playroom 운영 가이드

이 문서는 본 저장소를 **유아 프린터블 이미지 갤러리**로 일관되게 유지하기 위한 운영 기준입니다.

## 1) 핵심 목표

- 사이트 전반을 이미지 우선 구조로 유지
- 주제별 10장 이상 이미지 세트 제작
- 홈/포스트를 읽는 페이지가 아닌 보는 페이지로 유지

## 2) 페이지 원칙

- 홈: 큰 비주얼 타일 중심
- 포스트: 주제 라벨 + 이미지 그리드
- 버튼/링크: HOME, PRINT/DOWNLOAD 등 최소 UI만 유지
- 장문 설명, 긴 목록, 블로그형 본문은 제거 우선

## 3) 제작 파이프라인

1. 테마 선정 (예: 동물, 계절, 미로, 숨은그림)
2. 10컷 샷리스트 작성
3. 배치 생성 및 후보 선별
4. 최종 10장 이상 확정
5. 갤러리 페이지 삽입 및 짧은 캡션 부여

## 4) 품질 체크리스트

- [ ] 한 페이지에 이미지 10장 이상
- [ ] 캡션이 짧고 단순함
- [ ] 유아 친화 색감/형태 유지
- [ ] 시각적으로 복잡한 요소 최소화
- [ ] 페이지가 텍스트 중심으로 보이지 않음

## 5) 유지보수 원칙

- 현재 주제와 무관한 문서/설정/콘텐츠는 수정 또는 제거
- 재사용 가능한 이미지 워크플로우를 우선
- 새 주제를 추가할 때도 동일 규칙을 반복 적용
