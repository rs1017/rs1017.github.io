# Printable Playroom 운영 가이드

이 문서는 본 저장소를 **유아 프린터블 활동지 프로젝트**로 유지하기 위한 운영 기준입니다.

## 1) 핵심 목표

- 사이트 전반을 활동지 우선 구조로 유지
- 활동 유형별 10장 이상 이미지 세트 제작
- 홈/포스트를 읽는 페이지가 아닌 바로 고르고 출력하는 페이지로 유지

## 2) 페이지 원칙

- 홈: 큰 비주얼 타일 중심
- 포스트: 활동 유형 라벨 + 이미지 그리드
- 버튼/링크: HOME, PRINT/DOWNLOAD 등 최소 UI만 유지
- 장문 설명, 긴 목록, 블로그형 본문은 제거 우선
- 색칠공부를 제외한 페이지는 선화, 흑백, 최소 채색 우선

## 3) 제작 파이프라인

1. 활동 유형 선정 (예: 색칠, 숨은그림, 미로, 영어)
2. 10페이지 활동 목록 작성
3. AI 소스 생성과 레이아웃 조립 범위 분리
4. 최종 10장 이상 확정
5. Markdown 포스트 삽입 및 짧은 캡션 부여

## 4) 품질 체크리스트

- [ ] 한 페이지에 이미지 10장 이상
- [ ] 캡션이 짧고 단순함
- [ ] 활동 목적이 한눈에 보임
- [ ] 색칠공부를 제외하면 과도한 풀컬러가 아님
- [ ] 시각적으로 복잡한 요소 최소화
- [ ] 페이지가 텍스트 중심으로 보이지 않음

## 5) 유지보수 원칙

- 현재 주제와 무관한 문서/설정/콘텐츠는 수정 또는 제거
- 재사용 가능한 이미지 워크플로우를 우선
- 새 주제를 추가할 때도 동일 규칙을 반복 적용
- 단순 감상용 그림 세트는 방향에서 제외
