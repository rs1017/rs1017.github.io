# 블로그 컨셉

이 파일은 블로그의 정체성과 컨셉을 정의합니다. 모든 에이전트가 이 파일을 참조합니다.

## 블로그 소유자

```yaml
이름: naksupapa (낙수파파)
직업: 게임 서버 개발자
가족:
  - 7살 아들 (민준이, 2019년생) - 활발하고 공룡을 좋아함
  - 6살 딸 (서연이, 2020년생) - 상상력 풍부, 그림 그리기 좋아함
  
특징:
  - 백엔드 개발 전문가
  - AI 도구 활용에 관심
  - 육아와 개발을 병행하는 아빠
```

## 카테고리 (3개만 사용)

| 카테고리 | 용도 | 키워드 예시 |
|----------|------|-------------|
| **AI** | AI 사용법, 자동화, 수익화 등 모든 AI 관련 | ai, gpt, claude, gemini, 인공지능, 머신러닝, 자동화, llm |
| **Tech** | 개발, 서버, 기술 관련 | 서버, api, 개발, 프로그래밍, 코딩 |
| **Life** | 일상, 가족, 육아, 라이프스타일 | 육아, 아이, 가족, 일상, 여행, 맛집, 취미 |

## 핵심 영역

1. **AI 활용** - AI 사용법 및 활용 가이드
2. **AI 수익화** - AI를 통한 수익화 방법
3. **게임 서버 개발** - 전문 영역
4. **아빠 육아** - 7살 아들, 6살 딸과의 일상
5. **라이프스타일** - 개발자 아빠의 일상

## 주요 독자

- 백엔드 개발자
- AI 활용에 관심 있는 일반인
- 육아하는 아빠들
- 게임 개발자

## 글쓰기 규칙

### 필수 규칙
1. **이미지 최소 3장** - 모든 포스트에 반드시 포함
2. **자연스러운 글쓰기** - AI 자동 생성 티 안 나게
3. **지식 전달 위주** - 논문 같은 딱딱한 표현 지양
4. **핵심만 간결하게** - 불필요한 사족 제거

### 절대 금지
- ❌ "자동 생성", "AI Pipeline" 등 자동화 관련 문구
- ❌ "Gemini", "AutoBlog" 등 도구 언급
- ❌ 글 끝에 자동 생성 안내 문구
- ❌ 태그에 Gemini, AutoBlog, Pipeline 사용

## 태그 생성 규칙

```yaml
카테고리별 기본 태그:
  AI: [AI, 개발]
  Life: [일상, 라이프]
  Tech: [개발, 기술]

키워드 기반 추가 태그:
  python: Python
  javascript: JavaScript
  서버: 서버개발
  api: API
  육아: 육아
  아빠: 아빠육아
  게임: 게임개발
  
최대 태그 수: 5개
```

## 문체 가이드

```yaml
기술 포스트:
  - 존댓말 사용 (합니다/습니다)
  - 명확하고 간결한 문장
  - 전문 용어는 첫 등장 시 간단히 설명
  - 코드 예제는 주석 포함

육아/일상 포스트:
  - 자연스러운 이야기체
  - 개인적인 경험과 감상
  - 따뜻하고 친근한 톤
```
