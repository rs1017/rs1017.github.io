## 개요

Frontend Design은 독특하고 프로덕션급 프론트엔드 인터페이스를 높은 디자인 품질로 생성하는 스킬입니다. 일반적인 AI 스타일을 피하고 창의적이며 세련된 코드와 UI 디자인을 생성합니다.

## 디자인 사고 프로세스

코딩 전 컨텍스트 이해와 대담한 미학적 방향 설정:

1. **목적**: 이 인터페이스가 해결하는 문제는?
2. **톤**: 미니멀, 맥시멀리스트, 레트로-퓨처, 유기적, 럭셔리 등
3. **제약**: 프레임워크, 성능, 접근성
4. **차별화**: 무엇이 이것을 잊을 수 없게 만드는가?

## 미학 가이드라인

### 집중해야 할 요소

| 요소 | 가이드라인 |
|------|-----------|
| 타이포그래피 | Arial, Inter 대신 독특하고 특색 있는 폰트 선택 |
| 색상 & 테마 | CSS 변수로 일관성, 지배적 색상 + 날카로운 액센트 |
| 모션 | CSS 전용 솔루션 우선, 고영향 순간에 집중 |
| 공간 구성 | 비대칭, 오버랩, 대각선 흐름, 그리드 깨기 |
| 배경 & 디테일 | 그라디언트 메시, 노이즈 텍스처, 기하학 패턴 |

### 피해야 할 것

- 과도하게 사용된 폰트 (Inter, Roboto, Arial)
- 진부한 색상 (특히 흰 배경에 보라색 그라디언트)
- 예측 가능한 레이아웃과 컴포넌트 패턴
- 쿠키커터 디자인

## 사용 시점

- 웹 컴포넌트, 페이지, 아티팩트, 포스터, 애플리케이션 구축 시
- 웹사이트, 랜딩 페이지, 대시보드, React 컴포넌트 생성 시
- HTML/CSS 레이아웃 스타일링/미화 시
