## 개요

Internal Comms는 회사 내부 커뮤니케이션 작성을 위한 리소스와 가이드라인을 제공하는 스킬입니다.

## 지원 문서 유형

| 유형 | 설명 |
|------|------|
| 3P 업데이트 | Progress, Plans, Problems 팀 업데이트 |
| 회사 뉴스레터 | 회사 전체 뉴스레터 |
| FAQ 응답 | 자주 묻는 질문 답변 |
| 상태 보고서 | 프로젝트/팀 상태 보고 |
| 리더십 업데이트 | 경영진 대상 업데이트 |
| 프로젝트 업데이트 | 프로젝트 진행 상황 공유 |
| 인시던트 보고서 | 장애/사고 보고서 |

## 사용 방법

1. 요청에서 커뮤니케이션 유형 식별
2. `examples/` 디렉토리에서 적절한 가이드라인 파일 로드
3. 해당 파일의 지침에 따라 포맷, 톤, 콘텐츠 작성

### 가이드라인 파일

- `examples/3p-updates.md` - 3P 팀 업데이트
- `examples/company-newsletter.md` - 회사 뉴스레터
- `examples/faq-answers.md` - FAQ 응답
- `examples/general-comms.md` - 기타 커뮤니케이션

## 키워드

3P updates, company newsletter, weekly update, FAQs, internal comms, status reports, leadership updates
