## 개요

PPTX는 프레젠테이션(.pptx) 파일의 생성, 편집, 분석을 위한 종합 스킬입니다. 레이아웃, 코멘트, 발표자 노트 등을 지원합니다.

## 워크플로우

### 내용 읽기/분석

```bash
# 마크다운으로 변환
python -m markitdown path-to-file.pptx
```

### 새 프레젠테이션 생성 (템플릿 없이)

1. `html2pptx.md` 전체 파일 읽기
2. 각 슬라이드에 대한 HTML 파일 생성
3. html2pptx 라이브러리로 PowerPoint 변환
4. 썸네일 그리드로 시각 검증

### 기존 프레젠테이션 편집

1. `ooxml.md` 전체 파일 읽기
2. 프레젠테이션 언팩: `python ooxml/scripts/unpack.py`
3. XML 파일 편집
4. 유효성 검사: `python ooxml/scripts/validate.py`
5. 최종 패키징: `python ooxml/scripts/pack.py`

## 디자인 원칙

### 색상 팔레트 예시

| 이름 | 색상들 |
|------|--------|
| Classic Blue | Navy, Slate, Silver, Off-white |
| Teal & Coral | Teal, Deep teal, Coral, White |
| Black & Gold | Gold, Black, Cream |

### 레이아웃 팁

- 차트/테이블: 2단 레이아웃 또는 전체 슬라이드 레이아웃 권장
- 수직 스태킹 피하기 (가독성 저하)

## 썸네일 그리드 생성

```bash
python scripts/thumbnail.py template.pptx
# 기본: 5열, 그리드당 최대 30슬라이드
```
