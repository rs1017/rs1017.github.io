## 개요

DOCX는 Word 문서(.docx)의 생성, 편집, 분석을 위한 종합 스킬입니다. 추적 변경, 코멘트, 서식 보존, 텍스트 추출을 지원합니다.

## 워크플로우 결정 트리

| 작업 | 사용 방법 |
|------|----------|
| 내용 읽기/분석 | pandoc으로 마크다운 변환 |
| 새 문서 생성 | docx-js 사용 |
| 기존 문서 편집 | Document 라이브러리 (OOXML) |
| 법률/학술/비즈니스 문서 | Redlining 워크플로우 |

## 텍스트 추출

```bash
# 추적 변경 포함하여 마크다운으로 변환
pandoc --track-changes=all path-to-file.docx -o output.md
```

## Raw XML 접근

코멘트, 복잡한 서식, 문서 구조, 내장 미디어에 접근 시:

```bash
python ooxml/scripts/unpack.py <office_file> <output_directory>
```

### 주요 파일 구조

- `word/document.xml` - 메인 문서 내용
- `word/comments.xml` - 코멘트
- `word/media/` - 내장 이미지/미디어

## 새 문서 생성 (docx-js)

```javascript
import { Document, Paragraph, TextRun } from 'docx';

const doc = new Document({
  sections: [{
    children: [
      new Paragraph({
        children: [new TextRun("Hello World")]
      })
    ]
  }]
});
```

## Redlining 워크플로우

문서 리뷰를 위한 추적 변경 워크플로우:

1. 마크다운 표현 획득
2. 변경 사항 식별 및 그룹화
3. 문서 언팩 및 문서 편집
4. 검증 및 최종 패키징
