## 개요

Canvas Design은 디자인 철학을 기반으로 아름다운 시각 예술을 .png 및 .pdf 문서로 생성하는 스킬입니다.

## 워크플로우

```
┌─────────────────┐    ┌─────────────────┐
│ 1. 철학 생성     │───▶│ 2. 캔버스 생성   │
│ (.md 파일)      │    │ (.pdf/.png)     │
└─────────────────┘    └─────────────────┘
```

## 철학 예시

| 운동 이름 | 설명 |
|----------|------|
| Concrete Poetry | 기념비적 형태와 대담한 기하학을 통한 소통 |
| Chromatic Language | 색상을 기본 정보 시스템으로 사용 |
| Analog Meditation | 질감과 여백을 통한 시각적 명상 |
| Geometric Silence | 순수한 질서와 절제 |

## 핵심 원칙

1. **시각적 철학** - 디자인을 통해 표현할 미학적 세계관 생성
2. **최소한의 텍스트** - 텍스트는 희소하고 필수적으로만, 시각 요소로 통합
3. **공간적 표현** - 아이디어는 공간, 형태, 색상, 구성으로 소통
4. **전문가 수준** - 박물관/잡지 품질의 작품 추구

## 품질 기준

- 한 페이지의 고도로 시각적인, 디자인 중심 출력
- 반복 패턴과 완벽한 도형 사용
- 제한된 색상 팔레트로 의도적이고 응집력 있는 느낌
- 인간이 수많은 시간을 들여 만든 것처럼 보이는 품질
