# 프린터블 플레이룸 소개

> 집에서 바로 출력해 쓰는 유아 활동지 모음

---

## 이 블로그의 목적

프린터블 플레이룸은 **유아와 초등 저학년을 위한 출력형 활동지**를 정리하는 블로그입니다.

블로그의 중심은 긴 설명이 아니라, 아래처럼 **바로 써먹을 수 있는 종이 콘텐츠**입니다.

- 색칠도안
- 숨은그림 찾기
- 미로 찾기
- 따라그리기 / 선 긋기
- 오려붙이기 활동지
- 숫자 / 한글 / 알파벳 학습지

---

## 운영 원칙

- 모든 콘텐츠는 **A4 출력 기준**으로 생각합니다.
- 흑백 프린터에서도 보기 좋게 **선과 여백을 단순하게** 유지합니다.
- 글 하나를 읽고 끝나는 구조가 아니라, **미리보기 → 출력 → 바로 사용**으로 이어지게 구성합니다.
- 연령대와 난이도를 같이 적어 **부모와 교사가 바로 선택**할 수 있게 합니다.

---

## 주로 올라오는 콘텐츠

| 유형 | 예시 |
|------|------|
| **색칠도안** | 동물, 계절, 탈것, 바다 |
| **퍼즐** | 숨은그림, 미로, 짝 찾기 |
| **기초 활동지** | 곡선 따라가기, 점선 잇기 |
| **만들기 활동** | 오려붙이기, 장면 완성하기 |
| **학습지** | 숫자 세기, 한글/알파벳 따라쓰기 |

---

## 사용 대상

- 집에서 아이와 함께 출력해서 쓰려는 부모
- 수업 자료를 빠르게 찾으려는 교사
- 짧은 활동지를 반복적으로 활용하려는 돌봄/학습 공간
