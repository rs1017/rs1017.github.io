# 블로그 실행 로드맵 (우선순위)

이 문서는 블로그를 "초기 구축"에서 "지속 성장" 단계로 옮기기 위한 실행 항목을 정리한 체크리스트입니다.

## P0 (이번 주)

- [ ] 카테고리별 최소 1개 이상 글 발행
  - Server
  - AI
  - Database
  - DevOps
  - Architecture
- [ ] 주간 발행 리듬 확정 (예: 평일 1편)
- [ ] Google Analytics ID 설정
- [ ] 대표 글 1개를 상단 고정(시리즈 시작 글)

## P1 (다음 주)

- [ ] 댓글 정책 통일
  - 옵션 A: 댓글 비활성화 유지
  - 옵션 B: Disqus 활성화(shortname 설정)
- [ ] README를 "운영 문서"로 업데이트
  - 로컬 실행 방법
  - 포스트 작성 규칙
  - 배포 파이프라인 요약
- [ ] About 페이지에 최신 운영 방향 반영

## P2 (이번 달)

- [ ] 카테고리별 시리즈 기획
  - 예: "게임 서버 아키텍처 실전"
- [ ] SEO 기본기 강화
  - 제목 규칙/메타 설명/내부 링크/태그 표준화
- [ ] 월간 성과 리뷰 템플릿 도입
  - 유입 수
  - 상위 포스트
  - 평균 체류 시간
  - 다음 달 액션

## 운영 원칙

1. **발행량보다 일관성**: 무리한 대량 발행보다 꾸준한 리듬 유지
2. **기술 깊이 + 실무성**: 코드/설계/트러블슈팅 경험 중심
3. **측정 기반 개선**: 감이 아니라 데이터로 다음 주제 결정
